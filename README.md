# Forge Path

Forge Path is a platform where users document experiments while learning technologies and frameworks. Users can record setup details, outcomes, issues, and solutions. They can search experiments, compare them, and track their progress through XP, tiers, and badges.

## Getting Started

1. Set up MongoDB and ensure it runs on mongodb://127.0.0.1:27017 or config `MONGODB_URI` in `.env`.
2. Install dependencies: `npm install`
3. Run the application: `npm run dev`
4. Open the frontend from the `client/` folder.
